Ajuda rápida

Como usar o apontamento

01
Preencha os dados principais

Informe data, turno, nome do colaborador e matrícula.

02
Informe o TFM

Digite o número do TFM com 6 números e o código do projeto, se houver.

03
Anexe o PDF se existir

Use apenas PDF de até 5 MB. Se não tiver documento, o apontamento pode ser salvo sem anexo.

04
Registre as atividades

Selecione ou digite a atividade realizada, informe as horas e adicione observação quando necessário.

05
Salve o apontamento

Ao salvar, os dados vão para a planilha e o PDF, quando anexado, vai para o Google Drive.

06
Consulte um TFM

Use o campo Consultar TFM no topo para ver dados, atividades registradas e abrir o PDF quando existir.
